# Video scripts + shot lists

Devlog format. Every video has the same skeleton:

1. **Cold open (≤ 10s)** — a single concrete claim or screen.
2. **Context (30s)** — what we're building and why this video.
3. **Body (3–8 min)** — the actual work, mostly screen recording.
4. **Recap (15s)** — one sentence + the next-video CTA.
5. **End card** — `skymetric.dev` + `@skymetric` + Discord invite.

Every video gets a 60-second vertical "short" cut for YouTube Shorts,
TikTok, IG Reels. The short uses the cold open + the most concrete 40s of
the body.

## Recording setup ($0)

- **Screen:** OBS Studio (free, open-source).
- **Voice:** Whatever mic you have. Run audio through
  [Adobe Podcast](https://podcast.adobe.com/enhance) (free in beta) to
  clean it.
- **Editing:** DaVinci Resolve free tier — everything except 4K is unlocked.
- **Thumbnail:** Re-use [`../social/assets/post-square.svg`](../social/assets/post-square.svg)
  with edited copy for each episode. Export at 1280×720 (YouTube spec).
- **Captions:** Auto-generated via YouTube Studio, then hand-corrected.
  ~5 min per video.
